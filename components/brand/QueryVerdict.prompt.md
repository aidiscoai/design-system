MATCH / IGNORE verdict card — the brand's core narrative device: does the AI engine select this business, or pass it over?

```jsx
<QueryVerdict verdict="match" title="Optimised Business"
  reasons={["Structured data validated", "Authority signals high"]} />
<QueryVerdict verdict="ignore" title="Invisible Business"
  reasons={["Weak citation layer", "Unstructured content"]} />
```

Use the pair side-by-side for before/after comparisons. `match` reads green/positive, `ignore` reads negative.
